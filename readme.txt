== Installation ==

= Automatic download =

Visit the plugin section in your WordPress, search for [wpsustainable]; download and install the plugin.

= Manual download =

Extract the contents of the ZIP and upload the contents to the `/wp-content/plugins/wpsustainable/` directory. Once uploaded, it will appear in your plugin list.

== Frequently Asked Questions ==

= Where does the sustainability information come from? =

Data is coming from [The Green Web Foundation](https://www.thegreenwebfoundation.org/) API.
